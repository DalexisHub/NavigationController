# Lab 06 - UIKit Navigation

Este repositorio contiene los ejercicios de la segunda parte del laboratorio de navegación en UIKit.

## Ejercicios incluidos
- BasicNavigationController
- PassingDataBetweenViewControllers
- ModalPresentations
- Barra de Navegación Personalizada
- Navegación con Controlador de Barra de Pestañas

## Estructura
- `BasicNavigationController`: ejercicio 1
- `PassingDataBetweenViewControllers`: ejercicio 2
- `ModalPresentations`: ejercicio 3
- `Barra de Navegación Personalizada`: ejercicio 4
- `Navegación con Controlador de Barra de Pestañas`: ejercicio 5

## Tecnologías
- Swift
- UIKit
- Xcode
